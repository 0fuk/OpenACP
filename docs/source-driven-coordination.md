# Source-Driven Coordination

OpenACCP starts from local facts.

## Current Facts

Current sources can drive task cards and implementation.

## Reference Sources

Reference sources can explain history, constraints, or rejected alternatives. They do not expand current scope by themselves.

## Deprecated Sources

Deprecated sources must not drive current work unless explicitly promoted by the owner or final authority.

## Conflict Policy

When sources conflict:

1. Safety, security, and legal constraints win.
2. Current sources win over reference sources.
3. Explicit out-of-scope wins over plausible expansion.
4. Missing business meaning goes to the human owner.

## Reading Order

Read the shortest current path: project instructions, source pack, relevant spec or task card, then only the handoff or review needed for the current decision.
